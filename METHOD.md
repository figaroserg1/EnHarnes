# METHOD: короткая памятка по подходу

## Принцип
**Humans steer, agents execute**: человек задаёт рамки, агент исполняет и проверяет.

## 7 шагов
1. Зафиксируй цель и границы проекта.
2. Всё неизвестное пометь `TODO` с owner-маркером.
3. Сначала определи system spec, потом rules, потом architecture.
4. Настрой воспроизводимые скрипты (lint/build/test/tooling).
5. Дай агенту наблюдаемость (логи, проверки, артефакты).
6. Работай короткими PR и быстрыми follow-up.
7. Регулярно чисти документацию и закрывай устаревшие TODO.

## Definition of done для итерации
- Нет критичных TODO без owner-маркера.
- Все `EXAMPLE (REPLACE ME)` либо заменены, либо явно отложены в `docs/todo-registry.md`.
- `make lint` и `make build` проходят.
