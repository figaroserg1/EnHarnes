# TEMPLATE MAINTENANCE

Инструкции для задач, которые меняют сам этот template/scaffold.

## Правила работы
1. Не удаляй TODO, пока нет фактического содержимого.
2. Любой пример помечай `EXAMPLE (REPLACE ME)`.
3. Для каждого TODO указывай owner: `[HUMAN]`, `[AI]`, `[AI->HUMAN]`.
4. Изменения структуры синхронизируй с `ARCHITECTURE.md` и `docs/todo-registry.md`.

## Когда использовать
Используй этот файл, если меняешь:
- структуру директорий и навигационные документы;
- базовые policies/rules для всех будущих проектов на этом шаблоне;
- cross-cutting скрипты и инструменты в `scripts/`, `tools/`, `tooling/`.

## Связанные документы
- `AGENTS.md` — быстрая карта для агентов-пользователей шаблона.
- `ARCHITECTURE.md` — high-level карта модулей.
- `docs/todo-registry.md` — реестр незаполненного.
